Instalacja angular/cli
npm install -g @angular/cli@latest

Instalacja TypeScript
npm install --save-dev typescript @types/node @types/jasmine @types/core-js



Odpalenie lokalnego serwera
ng serve

Odpalenie tymczasowego serwera z przykładowymi danymi
json-server --watch ./api/data/db.json

Domyślny adres:
http://localhost:4200
